================================================================
   APP INFORMATION - CIRCLE MAP GENERATOR MOBILE
================================================================

**Application Name:** Circle Map Generator

**Version:** 2.0

**Package Name:** org.circlemapgen.circlemapgen

**Description:**
Create location-based circles with customizable radius, colors,
and opacity. Export to KMZ format for Google My Maps. Perfect
for coverage planning, distance visualization, and geographic
analysis.

================================================================
TECHNICAL SPECIFICATIONS
================================================================

**Framework:** Kivy 2.2.1

**Programming Language:** Python 3.7+

**Target Platforms:**
- Android 5.0+ (API 21+)
- iOS 11.0+
- Desktop (Windows/macOS/Linux) for testing

**App Size:**
- APK: ~20-30 MB
- IPA: ~25-35 MB

**Permissions Required:**
- Internet: For location search via OpenStreetMap API
- Storage (Read/Write): For saving KMZ files
- Optional Location: Future feature for GPS

**Minimum Device:**
- RAM: 1 GB
- Storage: 50 MB free
- Screen: 4" (320x480) or larger
- Android 5.0 / iOS 11.0 or higher

**Recommended Device:**
- RAM: 2 GB+
- Storage: 100 MB free
- Screen: 5" or larger
- Android 8.0+ / iOS 13.0+

================================================================
DATA & STORAGE
================================================================

**Input Data:**
- Location names (text)
- Coordinates (latitude/longitude)
- Circle parameters (radius, colors, opacity)

**Output Data:**
- KMZ files (Keyhole Markup Language, compressed)
- Saved to device storage
- Compatible with Google My Maps, Google Earth

**Storage Locations:**
- Android: /storage/emulated/0/Download/ or app folder
- iOS: App Documents folder
- Desktop: ~/Downloads/

**File Naming:**
- Format: circles_YYYYMMDD_HHMMSS.kmz
- Example: circles_20260215_143022.kmz

================================================================
NETWORK REQUIREMENTS
================================================================

**Internet Required For:**
- Location search (OpenStreetMap Nominatim API)
- Map tile loading (if map preview enabled)

**Works Offline:**
- Manual coordinate entry
- Circle configuration
- KMZ generation
- File sharing (local)

**API Usage:**
- OpenStreetMap Nominatim (free, no key)
- Rate limit: 1 request per second
- No authentication needed
