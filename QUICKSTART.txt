   QUICK START - MOBILE APP
================================================================

🚀 **Test on Desktop (Fastest)**

1. Install Kivy:
   pip install kivy Pillow requests

2. Run:
   python main.py

3. Test all features

================================================================

📱 **Build Android APK**

1. Install buildozer:
   pip install buildozer cython

2. Build (first time takes 30-60 min):
   buildozer android debug

3. Find APK:
   bin/CircleMapGenerator-2.0-debug.apk

4. Install on phone:
   - Copy APK to phone
   - Tap to install
   - Enable "Unknown Sources" if asked

================================================================

📖 **Using the App**

1. Search or enter location coordinates
2. Set radius and unit (meters/km/miles)
3. Choose colors and opacity
4. Tap "Add Circle"
5. Repeat for more circles
6. Tap "Generate & Save KMZ"
7. Upload KMZ to https://www.google.com/mymaps

================================================================

✨ **Key Differences from Desktop Version**

Desktop (Windows):
- Tkinter GUI
- Run with: python circle_map_gui.py
- No build needed
- Windows only

Mobile (Android/iOS):
- Kivy GUI
- Run with: python main.py (testing)
- Build APK/IPA for deployment
- Touch interface
- Cross-platform
